This program removes any specified residue from AMBER suite trajectory and topology files It also simultaneously produces radial distribution functions for the removed residue. This is intended to reduce the size of these files so quicker analyses can be performed. The purpose of producing an radial distribution is to extract any valuable information before the \hyperlink{classstrip}{strip}.
